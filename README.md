# House-price-prediction-project
## 📌 Overview
This project focuses on predicting house prices using machine learning techniques. It involves end-to-end data analysis, preprocessing, feature engineering, and model building to generate accurate price predictions.

---

## 🚀 Objectives
- Analyze housing data to identify key factors affecting prices  
- Build and evaluate machine learning models for prediction  
- Improve model performance using preprocessing and tuning techniques  

---

## 📊 Dataset
The dataset contains features such as:
- Location  
- Area (square feet)  
- Number of bedrooms and bathrooms  
- Other relevant housing attributes  

---

## ⚙️ Tech Stack
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn  

---

## 🔍 Project Workflow
1. Data Collection and Understanding  
2. Data Cleaning  
   - Handling missing values  
   - Removing duplicates  
3. Exploratory Data Analysis (EDA)  
4. Outlier Detection and Removal using IQR method  
5. Feature Engineering  
   - Encoding categorical variables  
   - Feature scaling  
   - Log transformation  
6. Model Building  
   - Linear Regression  
   - K-Nearest Neighbors (KNN)  
7. Model Evaluation  
   - Mean Absolute Error (MAE)  
   - Root Mean Squared Error (RMSE)  
   - R² Score  

---

## 📈 Results
- Built a predictive model with good accuracy  
- Linear Regression provided a strong baseline  
- KNN captured non-linear relationships effectively  

---

## 📌 Key Learnings
- Data preprocessing significantly improves model performance  
- Handling outliers enhances prediction accuracy  
- Feature engineering is crucial for better results
